# Задание 3. Применение условных вероятностей в анализе данных

## Описание набора данных

В данной работе используется набор данных Students Performance in Exams, полученный с платформы Kaggle.

Набор данных содержит информацию об успеваемости учащихся по трём предметам:
- математике,
- чтению,
- письму,

а также социально-демографические характеристики учащихся.
Назначение набора данных — анализ факторов, влияющих на результаты экзаменов, и демонстрация применения методов вероятностного анализа, в частности условных вероятностей.

## Описание признаков

Набор данных содержит следующие признаки:

- **gender** — пол учащегося (male, female)
- **ethnicity** — этническая группа
- **parental level of education** — уровень образования родителей
- **lunch** — тип школьного питания (standard, free/reduced)
- **test preparation course** — прохождение подготовительного курса (completed, none)
- **math score** — результат экзамена по математике
- **reading score** — результат экзамена по чтению
- **writing score** — результат экзамена по письму
